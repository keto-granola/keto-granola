# Keto Granola client

React client for keto granola e-commerce platform using:

- React
- Typescript
- Vite
- TanStack (queries)
- shadcn/ui (UI framework)
- Zod (validation)

## Development

### Prerequisite files:
- .env

### Setup:
```
npm i
```

### Run:
```
npm run dev
```

### Lint:
```
npm run lint
```
