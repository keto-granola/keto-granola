# Keto Granola client

React client for keto granola e-commerce platform using:

- React
- Typescript
- Vite
- TanStack (queries)
- shadcn/ui (UI framework)
- Zod (validation)

## Local Development

### Prerequisite files:
- .env

### Setup:
```
make dep
```

### Run:
```
make run
```

### Lint:
```
make lint
```

- Fix lint errors:
```
make lint/fix
```

- Check Typescript errors:
```
make typecheck
```

### Build:
```
make build
```
